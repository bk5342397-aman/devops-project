# DevOps Backend Project

## 📌 Overview
This is a beginner-friendly backend project built using Python and Flask. It demonstrates how to create a simple web server with basic API endpoints and understand core DevOps workflow concepts.

---

## 🚀 Features
- Simple Flask web server
- Homepage route (`/`)
- API endpoint (`/users`) returning JSON data
- Beginner-level DevOps project structure

---

## 🛠️ Tech Stack
- Python
- Flask
